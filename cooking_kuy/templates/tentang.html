Cooking Kuyy!

Cooking Kuyy! adalah sebuah program pencarian resep makanan menurut inputan gambar yang dipilih, resep yang dipilih, atau input kalimat yang mengandung unsur resep.

Program ini dibangun menggunakan bahasa pemograman Python untuk webservicenya dimana program juga memanfaatkan Flask framework sebagai penghubung program sistem cerdas dengan HTMLnya.
